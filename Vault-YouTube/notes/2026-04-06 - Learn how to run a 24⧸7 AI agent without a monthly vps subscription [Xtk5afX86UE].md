## Summary
- Demonstrates running a 24/7 AI agent on a Raspberry Pi instead of a monthly VPS, eliminating ongoing hosting costs.

## Key Points
- **The GCP problem**: Easy to set up, keeps local machine clean, but costs money every month—costs add up over time.
- **Raspberry Pi solution**: One-time hardware cost; zero monthly hosting fee; runs 24/7 at home.
- **Use case**: Persistent AI agent that never sleeps.
- **Tradeoff**: VPS is easier for beginners and more isolated; Raspberry Pi requires more setup but is cost-free to operate long-term.
