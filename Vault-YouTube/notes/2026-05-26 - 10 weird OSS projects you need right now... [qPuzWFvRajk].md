## Summary
- Showcases 10 unconventional open-source projects built by "humans who absolutely should not be allowed near a compiler"—proving creative developer culture is thriving beneath the AI slop layer.
- Highlights: Ratty (3D Rust terminal), Terminal Phone (Tor-based P2P calls via shell script), CUDA Oxide (Nvidia's Rust GPU kernel library), Wario Synth (Game Boy chiptune converter), and Honker (SQLite pub/sub).

## Key Points
1. **Ratty**: Terminal emulator built in Rust using Bevy game engine. Cursor is a spinning 3D rat. Ctrl+Alt+Enter tilts terminal in 3D space. Import custom Blender models. Eats 300MB RAM.
2. **Terminal Phone**: Open-source push-to-talk voice/text over Tor as a shell script. No servers, no accounts, no phone numbers. `.onion` address = your identity. End-to-end encrypted and ephemeral.
3. **CUDA Oxide**: Dropped by Nvidia on GitHub. Write GPU kernels in pure Rust with `#[kernel]` annotation—compiles directly to PTX. No C++ or FFI required.
4. **Wario Synth**: Paste a song → get a Game Boy chiptune. Uses Web Audio API (2 pulse waves + wave channel + noise channel). Runs entirely in browser.
5. **J Mail**: Gmail UI emulator as if you were Jeffrey Epstein using it.
6. **Epstein Exposed**: Searchable database of the Epstein files with network graph showing deep state connections.
7. **Xpedia**: Wikipedia as a TikTok feed. Downloads 40MB of Simple Wikipedia; infinite scroll feed; runs algorithm entirely in browser; no server-side processing.
8. **Pyrite**: Self-hostable in-browser desktop OS with taskbar, draggable windows, file manager, code editor, terminal.
9. **Honker**: SQLite extension (in Rust) that adds Postgres-style NOTIFY/LISTEN. Durable pub/sub, task queues, event streams, and cron scheduler—all in a single `.db` file.
10. **Hyperagent**: Build and deploy agents with their own browser, shell, and file system in isolated cloud sandbox. Sponsor—built research workflow for newsletter curation.
