# Voice Cloning

**What it is** — AI technology that creates a digital replica of a person's voice from audio samples. Once cloned, any text can be synthesized into audio that sounds like that person, with matching pitch, cadence, and timbre. Services like [[ElevenLabs]] expose this capability via API.

**Why it matters** — Enables personalized AI agents that respond in the owner's own voice (legitimate use: automated assistants), but also enables voice-based fraud and deepfakes — cloned voices are increasingly indistinguishable from real speech without forensic analysis.

**Example** — An [[OpenClaw]] agent configured with an ElevenLabs voice clone responds to Telegram messages in the owner's voice automatically, with no manual recording per response.
