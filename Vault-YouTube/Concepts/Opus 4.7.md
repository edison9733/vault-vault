# Opus 4.7

**What it is** — Anthropic's flagship large language model, released ~April 14, 2026. Key upgrades over Opus 4.6: **3.75 MP vision** (2,576 px long edge), **87.6%** on the SWE-bench software engineering benchmark, and improved aesthetic judgment described as "more tasteful and creative." Powers [[Claude Design]].

**Why it matters** — Sets the current top-end capability for Anthropic's model family; high-resolution vision is essential for design and document tasks. Community controversy: vocal users claim it regresses on general reasoning versus 4.6, with a theory that 4.6 was deliberately degraded pre-release to make 4.7 appear better. Still trails the unreleased "Mythos" model on SWE-bench. Noticeably slower than [[Google Stitch]] and Cursor Composer for generation tasks.

**Example** — Claude Design uses Opus 4.7 to read a Figma file at near-full pixel fidelity and generate 100 UI screens consistent with the uploaded design system, with working animations.
