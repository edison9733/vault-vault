# Botnet

**What it is** - A network of computers — often thousands or millions — that have been secretly compromised by malware and are controlled remotely by an attacker. The attacker uses them collectively to send floods of requests (DDoS), mine cryptocurrency, or distribute spam, all without the device owners' knowledge.

**Why it matters** - Botnets can generate traffic volumes no single server can withstand. A Distributed Denial-of-Service (DDoS) attack from a botnet can knock critical infrastructure offline even when each individual malicious request looks legitimate.

**Example** - On April 27 2026, a botnet hit GitHub's Elasticsearch subsystem, taking GitHub's code search offline for hours and affecting all 100M+ developers trying to search repositories.
