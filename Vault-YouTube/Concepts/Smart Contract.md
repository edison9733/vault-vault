# Smart Contract

**What it is** - A program stored on a blockchain that executes automatically when predefined conditions are met. The code is public, immutable once deployed, and runs on every node in the network — so no single party controls or can change it. On Solana, smart contracts are called "programs."

**Why it matters** - Smart contracts enable trustless agreements: two parties can transact without needing a bank or escrow service, because the code itself enforces the rules. They are the building blocks of [[DeFi]], NFTs, and most blockchain applications.

**Example** - A Solana CRUD program is a smart contract that lets users create and manage data entries on-chain. The [[Anchor Framework]] is the standard toolkit for writing these programs in Rust.
