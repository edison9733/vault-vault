# WordPress

**What it is** - A content management system (CMS) that powers approximately 43% of all websites on the internet. It lets non-developers build websites by installing themes and plugins — small add-ons that extend site functionality without writing code from scratch.

**Why it matters** - WordPress's massive ecosystem of ~60,000 plugins is both its greatest strength and its primary security liability: 96% of WordPress vulnerabilities originate from plugins, not the core system. The [[WordPress Plugin Architecture]] gives every plugin full server access with no sandboxing.

**Example** - In April 2026, a supply chain attack compromised 31 WordPress plugins by purchasing them on Flippa. Sites that received the routine plugin update had their `wp-config.php` modified, giving attackers full database and credential access — all through legitimate update channels.
