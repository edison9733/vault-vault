# GitHub

**What it is** - GitHub is a web platform built on top of [[Git]] that lets developers store code, track bugs (Issues), propose changes (Pull Requests), run automated tests ([[GitHub Actions]]), and collaborate publicly or privately. Think of it as "Facebook for code" combined with a deployment pipeline.

**Why it matters** - With 100M+ developers and 420M+ repositories, GitHub is effectively the public record of software. If a project isn't on GitHub, most developers won't find it. Microsoft acquired it for $7.5 billion in 2018.

**Example** - On April 23 2026, GitHub's [[Merge Queue]] silently unmerged 2,092 pull requests across 658 repos — illustrating that when GitHub fails, it fails at the core thing it exists to do: reliably store code.
