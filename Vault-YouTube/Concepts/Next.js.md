# Next.js

**What it is** - Next.js is a React-based web framework that adds server-side rendering, file-based routing, and built-in API routes on top of React. It runs on Node.js and lets you build both the front-end UI and back-end endpoints in a single project.

**Why it matters** - It is the most widely used full-stack React framework and the standard frontend layer in Solana dApp templates. Its combination of developer speed and production-grade deployment (via Vercel) makes it the default choice when scaffolding a new web3 app.

**Example** - The Solana Templates page offers a Next.js + Anchor template that wires wallet connection and smart contract bindings into a working dApp in a single scaffold command.
