# Solana Templates

**What it is** - An official collection of project scaffolds for building [[Solana]] dApps. Each template bundles a frontend framework (such as [[Next.js]]), a wallet adapter, and a smart contract toolkit (such as [[Anchor Framework]]) into a project that is ready to build and deploy immediately.

**Why it matters** - Templates eliminate the hours of manual wiring normally needed to connect a blockchain to a web UI. The goal is to compress the full create-build-deploy cycle to under 5 minutes, lowering the barrier for new Solana developers.

**Example** - Selecting the Next.js + Anchor template provides a working dApp scaffold with wallet connection and on-chain program bindings already configured — no boilerplate setup required.
