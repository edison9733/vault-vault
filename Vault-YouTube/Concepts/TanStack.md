# TanStack

**What it is** - A popular set of open-source JavaScript/TypeScript libraries for the React ecosystem, including TanStack Query (server state management), TanStack Table, TanStack Router, and others. Collectively downloaded tens of millions of times per week.

**Why it matters** - Its wide adoption made it a high-value target: compromising TanStack packages meant instant access to millions of developer machines and their credentials.

**Example** - In 2026, an attacker hijacked TanStack's release pipeline via a `pull_request_target` misconfiguration, publishing malware to 84 TanStack packages in a single attack.
