# Apache 2.0 License

**What it is** - A permissive open-source software license that allows anyone to use, modify, and distribute the software — including for commercial purposes — with no restrictions beyond attribution. "Apache 2.0" means complete freedom: you can build a product, sell it, and keep your changes private.

**Why it matters** - In the AI model space, many "open" models carry custom licenses that restrict commercial use or give the original company legal leverage. Apache 2.0 is the gold standard for genuine openness, which is why [[Gemma 4]]'s Apache 2.0 release was notable among frontier-class models.

**Example** - [[Gemma 4]] (Google) and OpenAI's GPT-OSS models are both Apache 2.0; Meta's Llama uses a special Meta license that gives Meta leverage over commercially successful users.
