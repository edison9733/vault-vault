# Anthropic

**What it is** — AI safety company founded in 2021 by former OpenAI researchers, including Dario Amodei and Daniela Amodei. Produces the Claude model family (Haiku → Sonnet → Opus tiers), with major releases including Claude Code (agentic coding tool), [[Opus 4.7]] (April 2026), and [[Claude Design]] (April 2026).

**Why it matters** — Positioned as one of the two primary rivals Google named in its I/O 2026 "interface to reality" race (alongside [[OpenAI]]). Anthropic's rapid product expansion — from language models into coding agents and UI design tools — signals a shift from API provider to end-user platform competitor, threatening incumbents like Figma.

**Example** — In April 2026, Anthropic released Claude Design powered by Opus 4.7; Figma stock dropped 7% within hours, illustrating how Anthropic's product moves now directly disrupt established software categories.
