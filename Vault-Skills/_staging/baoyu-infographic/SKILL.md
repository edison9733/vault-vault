# Infographic Generator

Generate professional infographics by combining **layout** (information structure) × **style** (visual aesthetics). Analyzes content, recommends combinations, and generates publication-ready infographics.

## Workflow

1. **Analyze content** — extract key themes, data points, and hierarchy
2. **Recommend layout×style** — based on content type and tone
3. **Present recommendation** — confirm with user before generating
4. **Generate infographic** — produce with appropriate image backend
5. **Deliver** — provide the output file with explanation
